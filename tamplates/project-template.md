## 📖 템플릿 사용법

1. 위의 템플릿 코드를 복사
2. 적절한 폴더에 새 파일로 저장
3. 템플릿 내용을 실제 내용으로 교체
4. 파일명 규칙에 따라 저장

- 프로젝트: `프로젝트명-YYYY-MM.md`
